# Agent Context

This folder stores durable context for agents working on the Adobe Premiere MCP
editor repo.

## Memory

Read `.agents/memory/premiere-workflow.md` before editing workflow code or
operating the live Premiere MCP path.

## Sessions

Session notes live in `.agents/sessions/`.

## Task Tracking

Use GitHub issues for tracked work. Do not create local `.agents/TASKS` or
`.agents/PRDS` folders.
