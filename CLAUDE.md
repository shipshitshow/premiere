# Claude Instructions

This repo is focused on Adobe Premiere Pro editing through Adobe MCP.

Use `apps/premiere` as the active workspace. The old self-editing Python app is
not part of this repo anymore; use git history only if it needs to be recovered.

Before changing Premiere workflow code, read:

- `.agents/memory/premiere-workflow.md`
- `apps/premiere/skills/premiere-mcp-ops/SKILL.md`

Never claim that a live edit succeeded only because an MCP tool returned
success. Verify the active sequence layout after cuts.
